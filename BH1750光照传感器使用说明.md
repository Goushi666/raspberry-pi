# BH1750光照强度传感器 - 树莓派Python使用指南

## 一、硬件连接

### 传感器接口说明
- 本传感器使用 **I2C 接口**通信
- 只需连接 **J1 接口**即可工作
- J2 接口为可选的中断引脚，基础使用不需要连接

### 接线方式（J1接口 → 树莓派）

| 传感器引脚 | 树莓派引脚 | 说明 |
|-----------|-----------|------|
| VCC | Pin 2 (5V) | 电源正极 |
| GND | Pin 6 (GND) | 电源地 |
| SCL | Pin 5 (GPIO 3) | I2C时钟线 |
| SDA | Pin 3 (GPIO 2) | I2C数据线 |

## 二、树莓派配置

### 1. 启用I2C接口
```bash
sudo raspi-config
```
选择：`Interface Options` → `I2C` → `Enable`

### 2. 安装系统工具与 Python 依赖
```bash
sudo apt-get update
sudo apt-get install i2c-tools
# 本项目驱动使用 smbus2（与下文逻辑一致，推荐虚拟环境内安装）
pip install smbus2
```
（亦可选系统包 `python3-smbus`，与 `smbus.SMBus` 对应；仓库实现为 `smbus2`，见 `src/sensors/bh1750.py`。）

### 3. 验证设备连接
```bash
sudo i2cdetect -y 1
```
应该能看到地址 `0x23`（或 ADDR 接高时的 `0x5C`）的设备

## 三、Python代码实现

### 完整代码
```python
import smbus
import time

# I2C配置
BH1750_ADDR = 0x23      # 传感器I2C地址
bus = smbus.SMBus(1)    # 树莓派I2C总线1

# BH1750指令
POWER_ON = 0x01                 # 上电指令
CONTINUOUS_HIGH_RES = 0x10      # 连续高分辨率模式

def init_sensor():
    """初始化BH1750传感器"""
    bus.write_byte(BH1750_ADDR, POWER_ON)
    bus.write_byte(BH1750_ADDR, CONTINUOUS_HIGH_RES)
    time.sleep(0.18)  # 等待传感器准备

def read_light():
    """读取光照强度值"""
    data = bus.read_i2c_block_data(BH1750_ADDR, 0x00, 2)
    lux = (data[0] << 8 | data[1]) / 1.2
    return lux

if __name__ == "__main__":
    print("BH1750光照传感器测试程序")
    print("按 Ctrl+C 退出\n")

    init_sensor()

    try:
        while True:
            lux = read_light()
            print(f"光照强度: {lux:6.2f} lux")
            time.sleep(0.5)  # 每0.5秒读取一次
    except KeyboardInterrupt:
        print("\n程序退出")
```
